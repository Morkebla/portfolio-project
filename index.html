Portfolio Projects

Assault

Assault" is one of the first projects I developed using Unity and C#. It's a timeline shooter game where the game pushes you forward towards enemies, and you have to shoot or avoid the targets.

A* Algorithm

The A* algorithm is a pathfinding algorithm that determines the shortest route from point A to point B. However, its implementation is more challenging than it seems, as it requires navigating through a complex web of logic. I applied this algorithm in a strategy game where units move along the designated path upon clicking, utilizing the nearest available route.

Castle Defence

I developed 'Castle Defence' during a C# course, employing the Breadth-First Search (BFS) algorithm for pathfinding. Players have the freedom to strategically place towers anywhere on the map. Should they impede enemy movement, the algorithm dynamically resets to find an alternative path.

Grid and Unit spawn

I challenged myself with something new by developing the grid and unit spawns for a game. In this game, players open a menu screen on the side, choose an icon to represent their character, with each icon corresponding to a unique character. Players can then place their chosen character anywhere on the playing board.

Frogwatch

I had the opportunity to collaborate with a team of six on a game jam, 'Frogwatch' made with unity and C# where my main focus was UI design. I was responsible for nearly every aspect of the game's UI, with only a few exceptions. I take great pride in this project, as it offers both fun and engaging gameplay.

Elephencing

In a team of four, we developed 'Elephencing' during a game jam using the Godot engine and C# language. My role extended to working on certain UI elements, as well as implementing a bone structure to enable trunk movement via mouse and keyboard controls.

Memory Game

As part of my training course with JustIT, I developed a memory game using JavaScript, CSS, and HTML. The game initially reveals the cards before flipping them over. Your task is to match similar cards, and you win the game by successfully matching all cards.

Weigh Different

This puzzle game, created for the GMTK Game Jam 2024, revolves around manipulating objects to adjust their sizes and observe how their weights change. Players use a scale to measure these weights and aim to determine the exact weight of each item by experimenting with different sizes and combinations. The gameplay encourages exploration and problem-solving in a fun, interactive environment.

Dapper Dasher

It is an obstacle game where obstacles spawn in your way and you have to jump over them developed entirely in C++ using Raylib library. I also personally implemented the collision system.
